# Boulder Defense

A tower defense game where you defend against giant boulders rolling down slopes. Built with HTML5 Canvas, TypeScript, and Vite.

## Features (Planned)

- **Enemies**: Boulders that accelerate on slopes (Normal, Metal, Magma).
- **Towers**: Cannon, Acid Sprayer, Disintegrator.
- **Traps**: Pit Traps, Jump Pads.
- **Systems**: Wave management, Tower upgrades, Special skills lottery.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
